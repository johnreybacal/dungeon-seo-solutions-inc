## Wildcards
Cartographer

Hidden in plain sight
 - Cloak of invisibility (become invisible when not moving to avoid enemy detection)

Bullet time
 - Triggering traps will trigger bullet time to avoid sudden death

## Dungeon exploration
 - Spawn at entrance door (first floor)
 - Trigger traps (or avoid if already mapped)
   - Survive
   - Place trap location on map
   - Types
     - Floor Spike
     - Trapdoor
     - Spike ball
     - Arrow
     - Flame thrower
 - Avoid enemy vision
   - Place enemy location on map
 - Find finish door
   - Place finish door location on map
   - Enter finish door to proceed to next floor
 - Go back to entrance door to go up the previous floor
 - Going back to entrance door of first floor saves progress
   - Dying loses progress
